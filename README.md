### Prerequisites

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/MuneebHussain52/mini-social-media.git
   ```

2. Navigate to the project directory
   ```bash
   cd mini-social-media
   ```

3. Install dependencies

   ```bash
   npm start
   ```

5. Open your browser at
   ```
   http://localhost:3000
   ```
